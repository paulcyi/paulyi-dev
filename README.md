## Getting Started

```bash
# Clone the repository
git clone https://github.com/paulcyi/paulyi-dev.git

# Install dependencies
cd paulyi-dev
npm install

# Run development server
npm run dev -- -p 3001

# Open http://localhost:3001 in your browser
```

- [Next.js](https://nextjs.org/) - React framework
